### 2. 用户态和内核态
> cpu分为不同的指令级别，Linux内核跑在Ring 0级；用户程序跑在Ring 3级，对于系统的关键访问，需要经过kernel的同意，保证系统健壮性  
>内核执行的操作 -> 200多个系统调用 sendfile read write  
>JVM：在OS的角度来看，就是一个普通程序
